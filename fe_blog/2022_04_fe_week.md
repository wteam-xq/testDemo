React 18 正式版发布发布，主要有以下更新：
新的客户端渲染 API ：ReactDOM.createRoot（用于替换 ReactDOM.render）。
服务端渲染 API 更新，完全支持了服务端的 Suspense 和流式 SSR。
自动批处理优化（解决了异步回调中无法批处理的问题）。

useId: 为组件生成唯一的 ID。

用于并发渲染的 useDeferredValue、startTransition、useTransition。

Android 13 开发者预览版 2 发布，支持了 COLRv1 彩色字体格式，并将系统表情符号更新成了 COLRv1 格式。COLRv1 是一种支持渐变、性能更好、体积更小的彩色字体格式，旨在用来取代 OT-SVG 格式。此前 Chrome 98 和 Edge 98 已经支持。更多请阅读《iconfont 支持全新的彩色字体图标》。
